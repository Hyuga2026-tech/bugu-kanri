# 武道具管理システム

武道具（小太刀・長剣等）の貸出・返却を管理するWebアプリです。

## 機能

- 物品一覧・状態管理（利用可能 / 申請中 / 貸出中 / 故障）
- 貸出申請（名前・目的・返却予定日）
- 管理者による申請の承認・却下
- 管理者による返却確認・状態変更
- Excelファイルからの物品一括インポート

## 管理者ログイン

パスワード: `Kumistral`

## ローカル開発

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチへのpushで GitHub Pages へ自動デプロイされます。
